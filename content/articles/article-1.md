# マークダウン記法まとめ

## 見出し2

### 見出し3

#### 見出し4

``` md
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
```

## 太字

太字: **太字**

``` md
**太字**
```

## リスト

- リスト1
- リスト2
  - リスト2-1

``` md
- リスト1
- リスト2
  - リスト2-1
```

## チェックボックス

- [ ] チェックボックス
- [x] チェックボックス

``` md
- [ ] チェックボックス
- [x] チェックボックス
```

## 引用

> 引用

``` md
> 引用
```

## リンク

[リンク](http://google.com)

``` md
[リンク](http://google.com)
```
